┌option┬─────────────────────────────────────────────────────────────────────────────┐
│  -a  │ 숨김 파일 폼함하여 조회할 경우 사용되는 옵션 입니다.                        │
│  -d  │ 디렉토리 정보를 확인할때 사용되는 옵션 입니다.                              │
│  -l  │ 파일 및 디렉토리 리스트를 상세하게 조회할 경우 사용되는 옵션 입니다.        │
│  -h  │ K, M, G 단위로 파일의 크기에 단위를 표시하여 조회되는 옵션 입니다.          │
│  -n  │ 파일 및 디렉토리의 소유자 및 소유자 그룹을 UID, GID로 조회하는 옵션 입니다. │
│  -R  │ 하위 경로의 파일을 모두 조회하는 옵션 입니다.                               │
│  -S  │ 파일의 크기 기준으로 조회하는 옵션 입니다.                                  │
│  -t  │ 변경시간을 기준으로 조회하는 옵션 입니다.                                   │
└──────┴─────────────────────────────────────────────────────────────────────────────┘
